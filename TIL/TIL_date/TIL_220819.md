### CASE문

- 특정 상황에서 데이터를 변환하여 활용
- ELSE 생략 시 NULL값 지정

```sql
CASE
	WHEN 조건식 THEN 식
	WHEN 조건식 THEN 식
	ELSE 식
END
```

- 예시

```sql
SELECT
	last_name,
	age,
	CASE
		WHEN age < 18 THEN '청소년'
		WHEN age <= 30 THEN '청년'
		WHEN age <= 64 THEN '중장년'
	END
FROM users
LIMIT 15;
```
