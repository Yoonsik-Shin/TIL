## Nano


![image-20230307163628338](Linux(3).assets/image-20230307163628338.png)

> `^` === `ctrl`

- 저장 : `ctrl + s`
- 파일 내보내기 : `ctrl + o`
- 파일 나가기 : `ctrl + x`
- 검색 : `ctrl + w`
  - 대소문자 구분 :  `alt + c`
