# Node JS

## Node REPL

- 시작하기 

```bash
$ node
```

- 종료하기
  1. `.exit`
  2. `ctrl + D`
  3. `ctrl + C`  두 번

```bash
.exit
```

- 명령어 도움말

```bash
.help
```

- `global`
  - 전역 객체로 모든 객체의 최상위 객체
  - JS에서의 `window`와 유사한 개념
  - global은 생략가능

```bash
global
```



## 파일 실행

- JS파일에서 동작함

```bash
node JS파일명.js
```
