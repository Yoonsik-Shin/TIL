  > 함수 미사용 시 표준편차 계산

  ```python
  values = [100, 200, 300]
  total = 0
  cnt = 0
  
  # 평균 계산
  for value in values:
      total += value
      cnt += 1
  mean = total / cnt
  
  # 분산 계산
  total_var = 0
  for value in values:
      total_var += (value - mean) ** 2
      
  sum_var = total_var / cnt
  target = sum_var
  
  # 표준편차 계산
  while True:
      root = 0.5 * (target + (sum_var/target))
      if (abs(root - target) < 0.00000000001):
          break
      target = root
      
  std_dev = target
  print(std_dev)
  ```

  

  > 내장함수 (Built-in Fuction) 활용

  ```python
  import math
  
  values = [100, 200, 300]
  mean = sum(values) / len(values)
  sum_var = sum(pow(value - mean, 2) for value in values) / len(values)
  std_dev = math.sqrt(sum_var)
  
  print(std_dev)
  ```
