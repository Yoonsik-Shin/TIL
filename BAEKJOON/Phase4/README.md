## 📑개념 정리 

```python
# f-string 포매팅
x = 3.141592653
f'{x:.3f}' #소수점 3자리까지 표현
>> '3.141'
f'{x:10.4f}' #소수점 4자리까지 표현 + 총 자릿수 10개 맞춤
>> '    3.1415'

f'{"hi":<10}' # 왼쪽 정렬
>> 'hi        ' 
f'{"hi":>10}' # 오른쪽 정렬
>> '        hi' 
f'{"hi":^10}' # 가운데 정렬
>> '    hi    ' 

f'{"hi":=^10}' # 가운데 정렬, =문자로 공백채우기
>> '====hi 
f'{"hi":!<10}' # 왼쪽 정렬, !문자로 공백채우기
>> 'hi!!!!!!!!'
```
